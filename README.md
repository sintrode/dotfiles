# dotfiles
Obligatory collection of dotfiles. They'll most likely change in the future. An attempt was made to strike a balance between minimalism and usefulness.

You are almost certainly going to have to change the interface name in .tmux.conf to the correct value, found in `ip addr show`.

Vim colorscheme is [palofobsidian by psmay](https://github.com/psmay/vim-colors-palofobsidian).
